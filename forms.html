Modules

These are the form elements available, place these within the starter template to build up your form.

In the starter template, the containers have an added '-form' attached to their class. This is for responsive purposes, and the sizes are the exact same. It is recommended to use at least 'container-half-form' as per source code for readability

You can place headings within forms

Starter Template - Forms

		
		
		
			<form action="#" method="post" class=""> <!-- Add Colour Here -->
				<!-- Add Content Here -->
				<!-- Add Content Here -->
				<!-- Add Content Here -->
			</form>

Text Input


		
		
		
		<div class="field-group">
			<label class="label" for="text">Text:</label>
			<div class="field">
				<input id="text" name="text" placeholder="Text" required="" type="text" value="">
			</div>
		</div>

Text Box


		
		
		
		<div class="field-group">
			<label class="label" for="textbox">Text Box</label>
			<div class="field">
				<textarea cols="50" id="textbox" name="textbox" placeholder="Text Box" rows="10" spellcheck="true"></textarea>
			</div>
		</div>
