Modules

These are the modules that will be placed within the containers you have placed in the starter template.

If you are using a Header, Navigation & Footer, they must be placed in their logical order shown below

You will see each modules respective attributes, if you view the source code you will see instructions and examples on how to change these

  1. Colors:
  2. Cyan (Default)
  3. Blue
  4. Pink
  5. Orange
  6. Yellow
  7. Purple
  8. White (Useful for text modules)
  1. Buttons:
  2. Square (Default)
  3. Rounded
  1. Icons:
  2. Font Awesome Icons
