Containers & Layout Options

Three layout options exist:

  1. 'container-default' takes up 33% of browser window
  2. 'container-half' takes up 66% of browser window
  3. 'container-full' takes up 95% of browser window

site > containers > modules

You can use up to 3 container-default's in sequence to build a 3 column layout.

You can use up to 2 container-half's in sequence to build a larger 2 column layout.

You can use 1 container-full to build a 1 column layout.

Use the code snippets below within the starter template in order to build your desired layout 

			
			
			
			<div class="container-default">
			
				<!-- 
				
				Column Width of 33% or 1/3 of browser window 
				
				3 of these in sequence will result in a 3 small column layout
				
				-->

				
			</div>
			
			
			
			
			
			
			
			<div class="container-half">
			
				<!-- 
				
				Column Width of 66% or 2/3 of browser window 
				
				2 of these in sequence will result in a 2 medium size column layout
				
				-->
				
				
			</div>
			
			
			
			
			
			
			
			<div class="container-full">
			
				<!-- 
				
				Column Width of 95% or 3/3 of browser window 
				
				1 of these will result in a 1 large size column layout
				
				--> 
			
			</div>
